I've always liked the idea of what I'm calling transparent text. I see this technique often, but it's usually done as a single image. It's where the letters, normally filled with a single color, are transparent, and allow a photo or other image to shine through. Some designers I talked to have referred to the effect as an alpha mask, text clipping, or blend mode. But to me, those terms don't really describe the final output. So transparent text it is ... for now

This effect is usually done in Photoshop. However, I wanted to be able to implement it with real HTML text, because accessibility is important. I discovered an implemention on CodePen, which I re-implemented above. The visual effect only works well on large text. This would be great design technique for a category or section header.

This technique uses the mix-blend-mode property in CSS. As of November 2015, we don't have full browser support yet.
